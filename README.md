# HDU-History-Killer

[![Python Version](https://img.shields.io/badge/Python-3.x-blue.svg)](https://python.org)
[![Flask](https://img.shields.io/badge/Flask-2.x-green.svg)](https://flask.palletsprojects.com/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个专为HDU中国近代史纲要课程设计的交互式复习软件，提供随机答题、章节练习、题目查询和错题解析功能，帮助用户高效备考。

## 📖 项目介绍

HDU-History-Killer 是一款基于 Flask 框架开发的历史复习 Web 应用，旨在帮助学生和历史爱好者系统地复习历史知识。应用包含大量历史选择题，支持多种学习模式，并提供详细的错题解析功能。

## ✨ 功能特色

### 🎯 核心功能
- **随机答题模式**：从题库中随机选择 10 道题目进行练习
- **章节练习模式**：选择特定章节的所有题目进行有针对性的复习
- **题目查询功能**：根据题干关键词快速查找相关题目和答案
- **分数结算系统**：答题结束后自动计算得分并显示详细结果
- **错题解析**：测试结束后显示错题及正确答案，帮助用户查漏补缺

### 📊 学习数据
- 实时计算答题准确率
- 提供详细的成绩分析
- 错题分类和统计
- 答题历史记录（待实现）

## 🚀 快速开始

### 环境要求
- Python 3.x
- pip 包管理器

### 安装步骤

1. **安装依赖**
   ```bash
   pip install flask
   ```

2. **启动应用**
   ```bash
   python app.py
   ```

3. **访问应用**
   在浏览器中打开：`http://127.0.0.1:5000`

## 📚 使用说明

### 首页
- 提供三个主要功能入口
- 显示应用简介和使用提示

### 随机答题
- 点击"随机答题"按钮开始练习
- 系统会随机选择 10 道题目
- 每道题目有四个选项，选择答案后点击提交
- 答题结束后查看得分和错题解析

### 章节练习
- 点击"章节练习"选择要复习的章节
- 系统会显示该章节的所有题目
- 题目会随机排序，点击提交后查看结果

### 题目查询
- 在搜索框中输入关键词
- 系统会返回包含该关键词的所有题目
- 点击题目可查看详细信息和答案


### 数据存储
- **JSON**：简单高效的数据存储格式
- **本地文件**：题库存储在 `历史选择题整理.json` 文件中

### 题库结构
```json
[
  {
    "id": 1,
    "chapter": "第一章",
    "question": "题目内容",
    "options": [
      {"letter": "A", "text": "选项A"},
      {"letter": "B", "text": "选项B"},
      {"letter": "C", "text": "选项C"},
      {"letter": "D", "text": "选项D"}
    ],
    "answer": "A"
  }
]
```

## 📝 API 接口

### 获取所有题目
- **路由**：`/api/questions`
- **方法**：GET
- **返回**：所有题目 JSON 数据

### 获取随机题目
- **路由**：`/api/random/<count>`
- **方法**：GET
- **参数**：count（题目数量）
- **返回**：指定数量的随机题目

### 查询题目
- **路由**：`/api/search/<keyword>`
- **方法**：GET
- **参数**：keyword（搜索关键词）
- **返回**：包含关键词的题目
